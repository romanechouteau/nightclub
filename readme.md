# install

# server
```
cd nightclub-server
npm install
npm run sequelize db:migrate
npm run sequelize db:seed:all
```

# client
```
cd nightclub-client
npm install
```

# run

# server
```
cd nightclub-server
npm start
```
http://localhost:3001/

# client
```
cd nightclub-client
npm start
```
http://localhost:8080/
